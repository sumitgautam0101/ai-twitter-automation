## Project

OpenSocial is an open-source, plugin-based AI social media posting automation tool (X/Bluesky/Mastodon), with niche-driven content sourcing, AI text/image generation, scheduling with anti-bot randomness, and Telegram approval workflow.

The full architecture plan (tech stack, project structure, data model, config format, pipeline, DB schema, and phased build order) is documented in [project.md](project.md). Phase 1 (MVP) implementation has not started yet — consult `project.md` Section 10 for build phasing before adding code.
